# Resto

# Sobre o projeto

Um site web de restaurante com os seguintes requisitos:
- Navbar com 6 tulos
-  Button Reservação
- Texto (Lorem)
- 8 Cards
- Um Carousel de imagen
- Um formulário com de contato com 3 campos

 # Tecnologias utilizadas
- HTML
- CSS
- JavaScript
